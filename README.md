# Superstore Sales Analysis

## Project Overview

This project analyzes Superstore sales data to uncover trends in sales, profit, customer purchasing behavior, and regional performance. Using Python for data cleaning and exploratory data analysis (EDA), SQL for querying data, and Power BI for visualization, the project provides actionable business insights through an interactive dashboard.

---

## Business Problem

Retail businesses need to monitor sales performance, profitability, and customer purchasing trends to make informed business decisions. This project helps answer key business questions and supports strategic planning.

This project addresses questions such as:
- Which product categories generate the highest sales and profit?
- Which regions perform the best?
- Which customer segments contribute the most revenue?
- What are the monthly and yearly sales trends?
- Which products have low profitability despite high sales?

## Key Insights

- Identified the top-performing product categories based on sales and profit.
- Analyzed monthly sales trends to identify seasonal demand patterns.
- Compared regional performance to identify high- and low-performing markets.
- Evaluated customer segments based on revenue and profitability.
- Built an interactive dashboard with filters for region, category, and time period.

---

## Business Recommendations

- Focus marketing efforts on high-profit product categories.
- Optimize pricing and discount strategies for low-profit products.
- Increase inventory for fast-selling products.
- Expand sales initiatives in high-performing regions.
- Review heavily discounted products to improve profitability.
